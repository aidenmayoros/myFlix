myFlix API Documentation:

Use URL Parameters Request URL example HTTP Method Request body format Response data
Get a list of all movies /movies N/A http://exampleurl/movies GET N/A A JSON object showing data of all movies.
Example:
Grab a movie by its name or title /movies/{title} title required http://exampleurl/movies/{title} GET N/A JSON object showing data about a single movie.
Example:
Get a list of all movies based off a genre /movies/genre/{genre} genre required http://exampleurl/movies/genre/{genre} GET N/A JSON object showing data about all found movies that share a genre type.
Example:
Get data about a director by name /movies//directors/{directorName} directorName required http://exampleurl/movies/directors/{directorName} GET N/A JSON object showing details of the found director.
Example:
Add a new user profile /movies/users N/A http://exampleurl/movies/users POST JSON
Example:
              {

Update a user's username /movies/users/{id}&username={username} id, username required http://exampleurl/movies/users{id28578234}&username={username} PUT N/A JSON obejct showing user and updated username.
Example:
Add a movie to users favorite list /movies/users/{id}/favorites/add/{movieTitle} id, movieTitle required http://exampleurl/movies/users/{id}/favorites/add&movie={movieTitle} PUT N/A JSON object showing added movie to favorite list.
Example:
Delete a movie from user's favorite list /movies/users/{id}/favorites&movie={movieTitle} id, movieTitle required http://exampleurl/movies/users/{id}/favorites/remove&movie={movieTitle} DEL N/A JSON object showing movie that was deleted from users favorites.
Example:
Delete a user's account /movies/users/delete&user={id} id required http://exampleurl/movies/users/delete&user={id} DEL N/A JSON object showing the user that was deleted.
